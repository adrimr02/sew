¿Que es Node.js?

Node es un entorno de ejecución multiplataforma para JavaScript que permite ejecutar codigo JavaScript fuera de un navegador, como una aplicación de escritorio más. Es principalmente usado para crear servidores web por su facilidad y rapidez, ya que utiliza un paradigma de programación orientada a eventos.

Esto ha provocado que muchas grandes empresas utilicen Node.js en su backend, como pueden ser Netflix, IBM, Microsoft, PayPal o LinkedIn, entre otras.

La historia de Node

Fue creado en 2009 por Ryan Dahl, en un principio para Linux y Mac OS X y no seria hasta 2011 que estaria disponible para Windows. Dahl creó Node debido a sus quejas sobre el servidor Apache, el más utilizado en la epoca para la creación de servidores web. Lo veía limitado a la hora de manejar muchas peticiones simultaneas.

En 2010 se introdujo npm un gestor de paquetes para Node. Este facilitaba mucho a los desarrolladores publicar, compartir y utilizar codigo para Node.js, distribuido en paquetes.

En 2015 se funda la Node.js Foundation, una organizacion sin animo de lucro que gestionaría y tomaria las decisiones para el desarrollo de Node.

El último gran acontecimiento ocurrió en 2019 cuando Node.js Foundation se unio a JS Foundation, otra organización que supervisaba decenas de proyectos de JavaScript de codigo abierto. Juntas formaron la OpenJS Foundation, que, además de controlar el desarrollo de todos esos proyectos, tiene le proposito de fomentar el uso JavaScript y otras tecnologías relacionadas.
